# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 必須ルール（絶対厳守）

### ワークフロー
- **絶対に勝手にコミット・プッシュしない** — 必ずユーザーの明示的な承認を得てから
- **新規ファイルをどんどん作らない** — 既存ドキュメントを先にチェックし、あれば更新する
- **スキルがある作業はスキル経由で行う** — 直接スクリプトを実行しない

### コミュニケーション
- **簡潔に答える** — 結論を先に、短く答える
- **応答言語は日本語**
- **コミットメッセージも日本語**

---

## スキル活用方針

データ調査やコマンド詳細は各スキルに定義されている。積極的にスキルを活用すること。

| 区分 | スキル | 用途 |
|------|--------|------|
| Git | `/commit` | 変更のコミット |
| Git | `/push` | リモートへプッシュ |
| Git | `/merge` | mainブランチへマージ |
| ドキュメント | `/slides-maker` | プレゼンテーション作成 |
